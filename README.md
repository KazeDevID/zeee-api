# zeee-api

Sedikit penjelasan tentang [zeee-api](https://github.com/KazeDevID/zeee-api).

## ```INSTALL```

> npm install zeee-api

## ``` UNINSTALL ```

> npm uninstall zeee-api

## ```Otakudesu```

``` 

const zeee = require('zeee-api');

const judul = 'kaifuku'

zeee.otakudesu(judul)

    .then(result => {

     console.log(result)

});

```
